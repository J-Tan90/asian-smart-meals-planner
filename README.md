# asian-smart-meals-planner
Assistant that plans meals based on available ingredients in the refrigerator and formulates a grocery list based on what's needed while taking into account multiple factors. 

Step 1: Understand user meal preferences and recipes from initial input.
Step 2: Formulate list of ingredients needed based on recipes and portion sizes.
Step 3: Take into account existing inventory of ingredients available at home. 
Step 4: Formulate grocery list based on multiple factors:
  1. Missing ingredients
  2. Items on sale (scraped from multiple online/offline sources)
  3. Items available / unavailable (from multiple online/offline sources + real-time user input)
Step 5: Adapt meal plan based on grocery items purchased
Step 6: Update inventory of existing ingredients after each use
